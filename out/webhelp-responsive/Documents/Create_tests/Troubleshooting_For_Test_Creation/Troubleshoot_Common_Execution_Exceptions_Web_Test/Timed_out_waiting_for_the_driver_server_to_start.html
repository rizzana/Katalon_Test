Timed out waiting for the driver server to start

Timed out waiting for the driver server to start.

  1. Download the correct Edge driver from the Microsoft website here: Microsoft Edge Driver based on your OS build (Go to Start > Settings > System > About and locate the number next to OS Build on the screen).
  2. Copy downloaded Edge driver and replace it in existing edgedriver folder of Katalon Studio. For example: C:\\Katalon\_Studio\_Windows_64-4.8\\configuration\\resources\\drivers\\edgedriver.
