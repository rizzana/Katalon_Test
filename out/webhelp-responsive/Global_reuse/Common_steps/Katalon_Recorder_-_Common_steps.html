Katalon Recorder - Common steps

  • Go to Workspace and select your test suite, then click Create a new Test Case.
  • Go to Workspace > Test Suites >Create a new Test Suite.
  • Go to Workspace > Test Data, and click Open a Data File. In the opened file dialog, select the data file your want to add.
  • Click on the Record button to start recording.
  • Click on the Play Test Case button to start executing the test case.
  • Click on the Play Test Suite button to start executing the test suite.
