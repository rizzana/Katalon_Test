Katalon TestOps - Common steps

  • Go to Planning > Releases.
  • Go to Planning > Requirements.
  • Go to Test Management > Test Cases.
  • Go to Test Management > Test Suites.
  • Go to Test Execution > Test Run List.
  • Go to Test Execution and click Schedule Test Run.
    The Schedule Test Run dialog pops up.
  • Go to Reports > Test Runs.
  • Go to Configurations > Script Repositories.
  • Go to Configurations > Test Environments.
  • Go to Configurations > Agent Setup.
  • Go to Configurations > Integrations.
  • Sign in to Katalon TestOps and go to your project.
    The project Dashboard appears.
  • Create a new project on Katalon TestOps:
    1. Go to TestOps > Settings > Project Management.
      The Project dashboard appears.
    2. Click on Create Project.
    3. Choose your team, then click Next.
      The Manage Projects page appears.
    4. Enter your project name, then click Create.
      You have successfully created a new project on Katalon TestOps.
  • Upload test scripts from the Git repository to Katalon TestOps:
    1. On Katalon TestOps, open your project.
    2. Go to Configurations > Script Repositories. Click Create Git Script Repository.
    3. Choose the Git repository source type, then input the Git repository URL, Git username and Git personal access token.
    4. Click Connect.
    5. Choose a Git branch, then input the project name and description.
    6. Click Create.
      You have successfully uploaded your test scripts from a Git repository to Katalon TestOps.
  • Log in to TestOps, then click Settings at the top right corner.
    From the dropdown menu, select User Management.
  • Select a test case by clicking on the test case name.
    The Test Case Detail page appears.
  • Select Edit to edit the test case details.
    The Edit Test Case panel appears.
